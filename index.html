ALT159.js - A Javascript library for basic types

String .capitalize() .numbers() .format(x) .underscore() .humanize() .pluralize() .singularize()

If you have ever worked with javascript, you probable found the methods on basic type lacking. ALT159 aims to fix this providing a simple to use collection of functions for your basic type.

ALT159 can be accessed by using the Javascrpit Function ALT159() or one of its two shortcuts f() or the windows key combination ALT+159 to get => ƒ

All source code for ALT159 can be found on Github

String functions

capitalize

capitalize the words in a string
ƒ('bacon').capitalize() => 'Bacon'

numbers

returns only the number in a string
ƒ('the meening of life: 42').numbers() => 42

format

a C style format method
ƒ('my name is {0}').format('Blarg') => 'my name is Blarg'

underscore

formats a string with underscores
ƒ('HoneyBadger dont care').underscore() => 'honey_badger_dont_care'

humanize

converts a string to be read by a human
ƒ('FluxCapacitor').humanize() => 'Flux Capacitor'
ƒ('1.21_giga_watts').humanize() => '1.21 giga watts'

pluralize

takes a word and pluralizes it
ƒ('person').pluralize() => 'people'
ƒ('Jelly Bean').pluralize() => 'Jelly Beans'
ƒ('mouse').pluralize() => 'mice'
ƒ('rice').pluralize() => 'rice'

singularize

takes a word and singularizes it
ƒ('kine').singularize() => 'cow'
ƒ('women').singularize() => 'woman'
ƒ('matches').singularize() => 'match'
ƒ('goats').singularize() => 'goat'
